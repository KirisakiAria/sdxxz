## 介绍
小圈子自嗨MUD游戏，正在开发中..

## 使用方法
运行dist文件夹中的index.html即可

## 运行命令
* npm run dev 使用webpack-dev-server在开发环境下运行项目
* npm run build 打包
* npm run start 使用electron运行项目
